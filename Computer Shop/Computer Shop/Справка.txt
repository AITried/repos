Данное приложение создано, чтобы работники нашего магазина могли легко взаимодействовать с базой данных.

- Слева сверху вы видите свои данные

- Ниже вы можете выбрать текущую таблицу, которая будет отображаться в самом низу окна

- Прямо над таблицей есть инструмент поиска для лёгкой ориентации в данных

- Правее поиска расположены кнопки управления данными:

Кнопка "Обновить" обновляет данные в таблице

Кнопка "Изменить" изменяет текщую выбранную ячейку таблицы

Кнопка "Удалить" полностью удаляет выбранную запись в таблице

Кнопка "Добавить" открывает новое окно, в котором можно добавить новую запись

- Для формирования отчёта можно нажать кнопку "Отчёт" справа сверху на панели
